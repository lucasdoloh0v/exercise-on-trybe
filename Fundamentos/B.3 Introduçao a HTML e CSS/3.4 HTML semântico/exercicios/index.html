Conhecendo tecnicamente:

O Stomatopoda (Odontodactylus scyllarus) é um animal de classificação:

Reino: Animalia
Filo: Arthropoda
Subfilo: Crustacea
classe Malacostraca
Subclasse Hoplocarida
ordem Stomatopoda

A visão do animal:

Enquanto os humanos tem receptores para 3 cores, o Stomatopoda tem receptores para 16 cores!

O golpe da lagosta-boxeadora!!

Esse animal pode desferir golpes de até 720km/h
